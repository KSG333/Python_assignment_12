## 주요 집합 연산
| 연산 명칭 | 연산자 (Operator) | 메서드 (Method) | 설명 |
| :--- | :--- | :--- | :--- |
| **합집합** (Union) | `세트1 \| 세트2` | `세트1.union(세트2)` | 두 집합의 모든 요소를 포함하는 새로운 집합 반환 |
| **교집합** (Intersection) | `세트1 & 세트2` | `세트1.intersection(세트2)` | 두 집합에 공통으로 존재하는 요소만 포함하는 새로운 집합 반환 |
| **차집합** (Difference) | `세트1 - 세트2` | `세트1.difference(세트2)` | 세트1에는 있고 세트2에는 없는 요소만 포함하는 새로운 집합 반환 |
| **대칭 차집합** (Symmetric Difference) | `세트1 ^ 세트2` | `세트1.symmetric_difference(세트2)` | 두 집합 중 어느 한쪽에만 존재하는 요소 (교집합 제외)를 포함하는 새로운 집합 반환 |

## Update 메서드
| 연산 명칭 | 연산자 (Operator) | 메서드 (Method) | 설명 |
| :--- | :--- | :--- | :--- |
| **합집합 업데이트(update)** | `세트1 \|= 세트2` | `세트1.update(세트2)` | 세트1에 세트2의 모든 요소를 추가 |
| **교집합 업데이트(intersection_update)** | `세트1 &= 세트2` | `세트1.intersection_update(세트2)` | 집합1을 두 집합의 교집합으로 변경 |
| **차집합 업데이트(difference_update)** | `세트1 -= 세트2` | `세트1.difference_update(세트2)` | 세트1에서 세트2와 겹치는 요소를 제거 |
| **대칭차집합 업데이트(symmetric_difference_update)**| `세트1 ^= 세트2` | `세트1.symmetric_difference_update(세트2)` | 세트1을 대칭 차집합으로 변경 |


## 집합 비교 연산
결과는 **`True`** 또는 **`False`**로 반환됩니다.
| 연산 명칭 | 연산자 (Operator) | 메서드 (Method) | 설명 |
| :--- | :--- | :--- | :--- |
| **부분집합** (Subset) | `현재세트 <= 다른세트` | `현재세트.issubset(다른집합)` | 현재 세트의 모든 요소가 다른 세트에 포함되는가? (두 집합이 같아도 True) |
| **진부분집합** (Proper Subset)| `현재세트 < 다른세트` | (메서드 없음) | 현재 세트가 다른 세트에 포함되면서, 두 세트가 서로 같지 않은가? |
| **상위집합** (Superset) | `현재세트 >= 다른세트` | `현재세트.issuperset(다른세트)` | 다른 세트의 모든 요소가 현재 세트에 포함되는가? (두 집합이 같아도 True) |
| **진상위집합** (Proper Superset)| `현재세트 > 다른세트` | (메서드 없음) | 다른 세트가 현재 세트에 포함되면서, 두 세트가 서로 같지 않은가? |
